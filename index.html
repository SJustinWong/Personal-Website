Not-Pictionary

An imitation of the popular game pictionary, created in JavaFX. The game is run on a multi-threaded server and can host up to 4 players. Players take turns rotating the role of "Drawer" and "Guesser"; 3 players take turns guessing while one attempts to draw one of three options provided to them from a list.

Made in collaboration with Deval Panchal, Daniel Earely, and Justin Katerberg.

Repo Link: https://github.com/jkaterberg/not-pictionary
